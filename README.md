# HmNormalizeFormKD

秀丸エディタ - 文字列正規化 - FormKD

今現在秀丸で表示中のフォントだと、存在しないか、もしくは、微妙に形が違う文字が表示されてしまう、などといった場合に、  
正規化して通常の文字に置き換えるためのマクロ＆プログラム。
